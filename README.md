# Statistical Modelling of Behavioural Data (R)

This project analyses tipping behaviour using statistical methods in R.

## Methods
- Exploratory data analysis and visualisation
- Linear regression
- Logistic regression

## Key Findings
- Total bill is a strong predictor of tip amount
- Group size has a moderate effect
- Variability increases with bill size (heteroscedasticity observed)

## Tools
- R (base)
